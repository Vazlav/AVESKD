Инструмент.

Консоль запросов by GtG.

Инструмент для работы с запросами с пользовательской стороны 1С.

Возможности:

Сохранение запросов в файл для последующего их использования.

Использование DBF файлов в качестве параметра запроса.

Вывод результата в виде табличного документа или таблицы значений.

Как пользоваться инструментом.

1) На первой закладке пишем запрос вручную или с помощью построителя.

2) Нажимаем кнопку Текст запроса в построитель. Делаем это каждый раз после изменения текста запроса.

3) На второй закладке задаем значение параметров.

4) На последующих закладках указываем параметры построителя если таковые используются в нашем запросе

5) Нажимаем кнопку Выполнить.

Особенности задания параметров.

Для того чтобы получить все использованные в запросе параметры нажимаем кнопку Параметры из запроса.

Заполняется таблица параметров.

Ддля простых параметров выбираем тип значения и указываем само значение.

Для параметров типа "список значений" справа вверху выбираем тип значений списка , ниже в поле списка выбираем значения, далее справа выбираем строку параметра которому нужно назначить список значений и нажимаем кнопку "Установить значение выбранного параметра запроса".

Для использования DBF файла в качестве параметра выбираем параметр слева и нажимаем кнопку "ДБФ Файл как параметр" В диалоге выбора файла выбираем файл. В результате значением параметра будет ДБФФайлВТипизированнуюТЗ("X:\YourFile.DBF"). При выполнении запроса файл из DBF перегоняется в таблицу значений с типизированными колонками с именами как поля в DBF файле. Такую ТЗ можно использовать в запросе в качестве параметра. Выбрать все ее содержимое во временную таблицу и далее использовать эту временную таблицу в связке с данными базы. Ограничение - результат выводится только в таблицу значений на закладку Результат.

Параметры можно сохранить в файл и восстановить из файла соответствующими кнопками.
