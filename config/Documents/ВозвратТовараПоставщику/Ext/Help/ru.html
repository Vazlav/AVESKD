Документ Перемещение товара

Используется для перемещения товара между складами

Технология работы:

1) Через обработку подбора по АП набирается необходимое количество товаров. В процессе подбора контролируется количество имеющееся на остатке склада.

Если товар ЖНВЛС или СВЛС требуется выбрать перемещаемую партию.
Цены и суммы склада отправителя расчитываются :

Для обычных товаров - исходя из имеющегося остатка на складе по усредненной стоимости

Для товаров ЖНВЛС/СВЛС - Исходя из данных хранящихся в партии

Цены и суммы склада отправителя не редактируются!

2) Если склад-получатель розничный - производится расценка товара

Если склад получатель оптовый - расценка не производится, ибо товар на оптовых складах учитывается в закупочных ценах

3) Документ проводится.
