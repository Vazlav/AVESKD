• Наименование - оно должно быть коротким, т.к. отображается на кнопке в справочной службе,
  • признак По умолчанию - может быть только у одной записи из справочника. Ггруппировка с установленным признаком по умолчанию используется при открытии справочной службы.
  • признак Для аутсорсинговой справки - может быть только у одной записи из справочника. Группировка с установленным признаком используется при открытии справочной службы сотрудниками сторонней справки.
  • Список брендов аптеки - список брендов аптек. При выборе в справочной службе группировки аптек показываются остатки аптек  у которых бренды содержатся в этом списке брендов. Один бренд может входить в разные группировки. Т.о. можно создать, например, группировку Дисконт, в которой будут аптеки брендов Аптека Эконом, горздрав, Горздарв А-фарм и группу Горздрав, в которой будут аптеки всех брендов относящихся только к горздраву.
ВНИМАНИЕ! Пометка на удаление группировки бесполезна. Если требуется удалить группировку её нужно удалять непосредственно.
