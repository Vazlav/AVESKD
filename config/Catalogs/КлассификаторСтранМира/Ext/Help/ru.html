Справочник "Классификатор стран мира"

Склады ® Классификатор стран мира

Справочник предназначен для хранения сведений о списке стран, используемых организацией. Страну можно указать, например, при описании гражданства физических лиц или при вводе импортной номенклатуры.

Список единиц измерения можно заполнить автоматически на основании общероссийского классификатора стран мира (ОКСМ) по кнопке Подбор из ОКСМ. В открывшейся форме нужно выбрать страну и добавить ее в список по кнопке ОК.
