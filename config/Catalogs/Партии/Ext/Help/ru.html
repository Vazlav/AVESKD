Справочник Партии

Используется для учета товаров  по партиям.

Позволяет определить закупочную цену каждой конкретной упаковки.

Цена госрегистрации - используется в расчете розничной цены при ценообразовании.

 

Партия-Родитель - Заполняется при переоценке товара. Это ссылка на исходную партию  товара которая была переоценена. Представляет собой  копию партии родителя, но с документом поступления Переоценкой (или другим, образовавшим новую партию)

Подчинен справочнику Ассортиментный план.
